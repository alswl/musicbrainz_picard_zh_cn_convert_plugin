# zh_CN Convert Plugin

Useage:

copy three .py files to plugin folder.

zh_TW to zh_CN by default.

You can edit `zh_cn_convert.py` `TO_LOCAL` to `zh-hant`, and will convert
Simple Chinese to Traditional Chinese.
